# Fun_Projects 🛠️

A collection of small, side projects — mostly built with Claude or other AI coding tools.

Some projects may be works in progress, and that's totally fine. The goal here is to explore, experiment, and have fun.

These are **"slide" (side) projects** — quick builds, experiments, and ideas that don't always need to be production-ready. Expect a mix of:

- 🚀 Finished (enough) projects
- 🔧 Works in progress
- 🤖 AI-assisted code and experiments

## Contributing / Using

Feel free to browse, fork, or get inspired. If something is broken or incomplete, it's probably intentional (or a great learning opportunity).
